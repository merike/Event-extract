Event-extract by merike

Extension

You can download a recent version of the extension here

Expected behaviour

For events extract both start and end date and time from email.

For tasks extract both date and time of due date from email and use email Date header as start time.

Language chosen depends on dictionaries installed. A dictionary that matched the highest amount of email content determines the language. If no dictionary gives a match of > 50% or no dictionaries are installed then application locale determines language used.

Testing

You don't need to have invitations to test. You can write a draft invitation yourself in any language you wish. To add as an event or task right-click the email/draft and choose "Add as".
